# Lite-wing_Drone
The Lite-Wing Drone project aims to develop an IoT-enabled unmanned aerial vehicle (UAV) capable of assisting in wildfire detection and environmental surveillance. The drone is equipped with a smoke sensing module and a wireless telemetry system that enables real-time monitoring of flight parameters and environmental conditions.
The primary objective of this work is to integrate an MQ2 smoke sensor with the Lite-Wing drone platform and transmit sensor readings along with flight telemetry to a ground monitoring station. The transmitted data can be visualized through a dashboard for monitoring and analysis.
This manual serves as a reference document for future interns and researchers working on the Lite-Wing Drone platform. It describes the hardware setup, firmware modifications, dashboard implementation, testing procedures, and troubleshooting methods developed during the project.
